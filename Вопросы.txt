a. Что такое jQuery?
jQuery — набор функций JavaScript, фокусирующийся на взаимодействии JavaScript и HTML. Библиотека jQuery помогает легко
получать доступ к любому элементу DOM, обращаться к атрибутам и содержимому элементов DOM, манипулировать ими. Также
библиотека jQuery предоставляет удобный API для работы с AJAX.

b. Преимущества jQuery?
-Существенно уменьшается количество кода (необходимого для работы скрипта) по сравнению с JavaScript, что в свою очередь
означает меньше временных затрат и более читабельный код
-Намного проще понять код (в отличии от JavaScript)
-Облегчает манипулирование моделью документа DOM
-Множество различных эфектов
-Использование Ajax становится намного проще
-Кросс-браузерная совместимость
-Огромное количество плагинов, с помощью которых можно сделать практически все что угодно
-Простота использования
-Очень удобная документация и активное комьюнити, готовое всегда оказать помощь при необходимости
