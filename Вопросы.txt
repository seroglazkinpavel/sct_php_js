a. В чем отличие html() и text()?
Разница заключается в способе получения информации. Если использовать функцию html (),
то значения из объекта получаются вместе с HTML кодом, который мы можем использовать как обычный HTML
для размещения его на странице. Если использовать text (), то мы получаем лишь текст из объекта без HTML-кода

b. Что будет если с помощью метода text() вставить текст с html тегами?
Получаем лишь текст из объекта без HTML-кода
